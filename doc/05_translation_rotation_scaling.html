translation, rotation, scaling

  1. translation

     
    xxxxxxxxxx
    x' = x + dx;
    y' = y + dy;
    z' = z + dz;

  2. rotation

     
    xxxxxxxxxx
    x' = x * cosB - y * sinB;
    y' = y * cosB + x * sinB;
    z' = z;

  3. scaling

     
    xxxxxxxxxx
    x' = x * sx;
    y' = y * sy;
    z' = z * sz;
